TOPIC

SHORT DESCRIPTION
    PowerShell module for working with CosmosDB.

LONG DESCRIPTION
    This module provides cmdlets for working with Azure Cosmos DB databases, collections,
    documents, attachments, offers, users, permissions, triggers, stored procedures
    and user defined functions.

EXAMPLES
    PS C:\> Get-Command -Module CosmosDB
